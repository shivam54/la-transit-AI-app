🚌 3. Routes for Agency

Get all routes for a specific agency

🚌 4. Route Details

Endpoint: https://api.metro.net/agencies/{agencyId}/routes/{routeId}?key={apiKey}

Get details for a specific route

🚌 5. Stops for Route

Endpoint: https://api.metro.net/agencies/{agencyId}/routes/{routeId}/stops?key={apiKey}

Get real-time vehicle locations for a route
