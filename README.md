## LA Transit App (V2)

LA transit application with an AI-powered assistant, basic navigation, and location-aware features.

This repository contains:

- A lightweight web prototype
- A Flutter mobile app


### API Keys and Security

- All real API keys have been removed and replaced with placeholders.
- Configure your own keys only in local environment variables or untracked config files.
- Do **not** commit any real secrets to this repository.
